O projekcie
Blog o tematyce najpopularniejszych diet. Połączony z bazą danych MySQL przy użyciu Xamppa. Umożliwia tworzenie, edytowanie, usuwanie oraz wyświetlanie postów.

Zastosowane technologie:
-HTLM
-CSS
-JAVASCRIPT
-EXPRESS
-MYSQL
